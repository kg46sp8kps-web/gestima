# QA Specialist Role

## Povinnosti

- spustit relevantní testy
- ověřit build/lint
- reportovat FAIL/PASS s důkazem
- explicitně uvést testovací mezery

## Report formát

- Backend tests: PASS/FAIL
- Frontend lint: PASS/FAIL
- Frontend build: PASS/FAIL
- E2E (pokud relevantní): PASS/FAIL/SKIPPED
- Rizika a mezery
