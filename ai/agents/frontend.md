# Frontend Specialist Role

Scope: `frontend/`.

## Checklist

- TypeScript strict bez `any`
- API volání přes `api/` moduly
- `data-testid` na interaktivních elementech
- bez inline anti-patternů (`!important`, hardcoded hex, Options API)
- komponenta je zapojená do parentu/routeru

## Quick commands

```bash
npm run lint -C frontend
npm run build -C frontend
bash scripts/ai/quality-gate.sh fe
```
