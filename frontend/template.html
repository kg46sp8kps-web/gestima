01 Filosofie designu

3 barvy

Cerna. Cervena. Sediva. Vse ostatni je informacni (status).

Informace > dekorace

Barva neni dekorace. Kazda barva musi nest informaci. Zelena = OK, cervena = smazat/chyba, zluta = pozor. Jinak seda.

Flat + blur

Zadny 3D, zadne gradienty. Hloubka pres backdrop-filter blur, jemne shadows a bordery. Moderny, cisty.

16 Pravidla (ke schvaleni)

SCHVALENO

  • Paleta: cerna + cervena + sediva. Status barvy (ok/error/warn) POUZE pro tecky a levy border.
  • Font: Space Grotesk (UI) + Space Mono (cisla, ceny, kody)
  • Buttons: 3 varianty — primary (brand), secondary (ghost), destructive
  • Icon Buttons: Default sedy, hover bily. Brand/danger POUZE pro specialni akce.
  • Badges: Monochromaticke (sedy bg) s barevnou teckou pro status
  • Toasty: Sedy bg, barevny levy border (3px)
  • Focus: Bily/svetle sedy ring — subtilni, nerusici
  • Selected row: Neutralni white-tinted rgba(255,255,255,0.06)
  • Edit mode: Svetlejsi pozadi (bg-raised) + silnejsi border
  • Price bar: Barevne segmenty (dataviz = jedine misto s multi-color)
  • Min font: 13px (zadny text mensi nez --text-xs, --text-2xs = 11px pro specialni pripady)
  • Inputs: Flat select (appearance: none), no spinners na number
  • Layout: Split-pane (ListPanel | DetailPanel) ve FloatingWindow
  • Container queries (NE media queries)
  • Backdrop blur na sticky header/nav

ZAKAZANO

  • Barevne pozadi na badges (badge-success, badge-danger, badge-warning, badge-info)
  • Barevne tlacitka krome brand a destructive (zadne btn-success, btn-warning, btn-info)
  • Barevne icon buttony krome brand a danger (zadne icon-btn-success, icon-btn-warning)
  • Native 3D select, number spinners
  • Cerveny/modry focus ring (pouzit bily subtilni)
  • Cervene zvyrazneni selected row (pouzit neutralni sedy)
  • Cerveny ramecek na edit mode (pouzit svetlejsi bg)
  • Font mensi nez 11px
  • Media queries (pouzit container queries)
  • Views pro features (pouzit Modules ve FloatingWindow)
  • Duplicitni barevne systemy (theme.css, variables.css) — POUZE design-system.css
