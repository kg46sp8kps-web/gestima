16 Design pravidla (ke schvaleni)

SCHVALENY standard

  • Font: Space Grotesk (UI text) + Space Mono (cisla, ceny, kody) — latin-ext pro CZ diakritiku
  • Flat inputs: appearance: none na select + no spinners na number input
  • Icon buttons: <IconButton> komponenta pro VSECHNY icon akce
  • Icon sizes: SMALL=16, STANDARD=20, LARGE=24, XLARGE=32
  • Text buttons: .btn-primary / .btn-secondary pro modaly a formulare
  • Focus: MODRY ring (#2563eb) - NIKDY cerveny!
  • Selected row: state-selected = rgba(153,27,27,0.15)
  • Edit mode: Red glow border (shadow-glow)
  • Numbers/ceny: font-mono + text-align: right
  • Layout: Split-pane (ListPanel | DetailPanel) uvnitr FloatingWindow
  • Dialogs: Horizontal layout (icon vlevo, text vpravo)
  • Container queries: container-type: inline-size (NE media queries!)

ZAKAZANO

  • Hardcoded icon sizes (:size="15", :size="18")
  • Custom button tridy (.action-btn-custom, .toolbar-btn)
  • Cerveny focus ring (focus = MODRY!)
  • Media queries pro responsive (pouzit container queries)
  • Views pro nove features (pouzit Modules ve FloatingWindow)
  • Native 3D select (appearance) — vsechny selecty musi byt flat
  • Sipky na number inputech (spinners) — -webkit-appearance: none
  • Duplicitni barevne systemy (theme.css, variables.css) — POUZE design-system.css
