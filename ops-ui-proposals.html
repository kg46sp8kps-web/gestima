Varianta A
Spreadsheet Grid
Buňky navigovatelné šipkami. Enter/F2 = edituj buňku. Tab = přejdi dál a ulož. Esc = zrušit. Mode = tlačítka místo dropdownu.

Navigace
  • ↑ ↓ ← → — pohyb po buňkách  ·  Enter nebo F2 = začni editaci
  • Tab = ulož buňku + přejdi na další  ·  Shift+Tab = zpět
  • Esc = zahoď změnu, vrať původní hodnotu
  • Pracoviště: Enter otevře typeahead popup (ne nativní select)
  • LOW / MID / HI jsou tlačítka — kliknutí nebo klávesy 1/2/3 na aktivní buňce
Varianta B
Row Expand (Master–Detail v tabulce)
Tabulka je vždy čitelná. Enter nebo klik na řádek rozevře editaci přímo pod ním. Tab projede pole. Enter = ulož + zavři. Esc = zahoď.

Přehled + detail
  • Tabulka je vždy celá viditelná — kdykoliv víš kde jsi
  • Klik nebo Enter na řádek = rozevře editaci pod ním (jen jeden najednou)
  • Tab / Shift+Tab projede všechna pole v rozvinuté sekci
  • Enter = uložit a zavřít  ·  Esc = zahodit  ·  klik mimo = zavřít
Varianta C
Flow Mode (Aktivní řádek)
Vždy jeden aktivní řádek s inputy. Ostatní jsou read-only. Tab jede polem po poli. ↓ nebo Enter z posledního pole = uloží + přejde na další. Nový řádek se vytvoří automaticky.
Tab= další pole
↑↓= přejít na řádek
↵= uloží + další řádek
Esc= zahodit
N= nová operace

Flow logika
  • Aktivní řádek = viditelné inputy. Ostatní = čitelný přehled.
  • Tab projede: Název → Pracoviště → Seřízení → Výroba → Mode (loop)
  • ↓ nebo Enter z posledního pole = uloží a aktivuje další řádek
  • Enter z posledního řádku = vytvoří nový prázdný řádek a přejde na něj
  • Esc = zahodit změny a deaktivovat řádek
Varianta D
Compact Cards (Vim-style)
Každá operace je vizuální karta. Klávesy J/K navigují mezi nimi. E = editovat, A = přidat, D = smazat. Aktivní karta se rozevře. Ostatní jsou přehledné.
J= dolů
K= nahoru
E= editovat
A= přidat
D= smazat
↵= uložit
Esc= zavřít

Klávesy
  • J / K — pohyb mezi kartami (vim-style, bez myši)
  • E — otevře editaci, Tab projede pole, Enter = uložit
  • A — přidá novou operaci za aktuální a přejde na ni
  • D — smazat (s konfirmací)  ·  Esc — zavřít editaci
  • Přehled je vždy viditelný — expanduje jen aktivní karta
