Logika
  • V klidu = čistá tabulka, žádný vizuální šum
  • Hover na řádek = podtržení polí signalizuje "editovatelné"
  • Klik / Tab na pole = plný focus s accent underline + světlý text
  • WC chip: klik = popup typeahead (ne nativní select)
  • Mode badge: klik cykluje LOW → MID → HIGH, auto-uloží
  • Tab průchod: Název → WC → Seřízení → Výroba → Mode → další řádek
  • Chevron ▶ vlevo = připraveno pro collapsible detail (neaktivní)
